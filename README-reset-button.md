# MiikuLive Distance Reset Button

Erillinen pyöreä nollausnappi Distance-widgetille.

Tätä kannattaa käyttää IRL Prossa omana browser overlayna ja asettaa näkyväksi vain itselle.

## Lataa nämä Distance-Widget-repoon

- reset-button.html
- reset-button.css
- reset-button.js

Normaali matka-widget jää ennalleen.

## Osoite

Kun tiedostot on ladattu Distance-Widget-repoon:

`https://miikueli.github.io/Distance-Widget/reset-button.html`

Pieni nappi:

`https://miikueli.github.io/Distance-Widget/reset-button.html?size=small`

Iso nappi:

`https://miikueli.github.io/Distance-Widget/reset-button.html?size=big`

## Tärkeää

Nappi nollaa saman päivän matkan localStoragesta avaimesta:

`miikuliveDistanceKm_YYYY-MM-DD`

Koska GitHub Pages on samalla domainilla `miikueli.github.io`, nappi ja Distance-widget pääsevät samaan selaimen muistiin.
